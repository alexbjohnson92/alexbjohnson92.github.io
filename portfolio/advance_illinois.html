The State We're In

Print/Digital Report

Advance Illinois’ objective was to create both a print and digital experience for their biannual report, The State We’re In. The State We’re In details public education in Illinois with the intent of informing, educating, and communicating current academic statistics and milestones to Illinoisans.

Multiple’s goal was to design a visual system integrating web graph animations that clearly and concisely communicated the data from the biannual report which would ultimately allow for Illinoisans to easily digest and visualize statistics in a more engaging and interactive way.
